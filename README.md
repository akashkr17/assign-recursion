## Recusion and Log
#### Introduction
This project is used to find factorial of a integer number using Head Recursion and 
Tail Recursion.
It also prints the log messages.


#### Requirements
###### 1.Scala 2.13.5
###### 2.sbt 1.5.5

#### Commands
##### clean

This command cleans the sbt project by deleting the target directory. The command output relevant messages.

```bash
sbt clean 
```
##### compile 
This command compiles the scala source classes of the sbt project.
```bash
sbt compile
```

##### tests

```bash
sbt test
```
